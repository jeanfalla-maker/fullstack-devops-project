# Full Stack DevOps Project

Proyecto de automatización en Bash para desplegar una aplicación Flask + PostgreSQL usando:
- Docker
- Docker Compose
- Kubernetes
- Docker Hub
- GitHub

## Requisitos
- Docker instalado
- Docker Compose habilitado
- kubectl instalado
- Un clúster Kubernetes activo (Minikube, Kind o similar)
- Git instalado
